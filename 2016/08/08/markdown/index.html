0x02 sublime text3 代码段

提高效率会让工作轻松，那么在写博客中使用markdown语法，每次都需要换行打符号，复制粘贴，这功夫snippet插件的自定义编辑代码段可以解决。

这里只是简单写一些自己现在开始用的代码，慢慢发现和创造其他的，是明明如月推荐给我使用的，并作了详细介绍，我觉得很不错，实用。

格式保存为xxx.sublime-snippet存放在C:\Users\Administrator\AppData\Roaming\Sublime Text 3\Packages\User

~1~snippet的组成部分功能:

1.content标签中必须有<![CDATA[…]]>，…替换成你需要的code或者某些字符,${1:this}表示按下tab键后光标所在位置，再次tab会调到${2:snippet}
2.<tabTrigger> 标签是自定义缩写字符串，意思就是设置成hello，当你打上hello接着按tab 就会替换成hello，this is a snippet

3.<scope>标签是表示在哪种语言环境下激活。写博客中可注释，只使用1和2的功能。
